This repository includes the source code of "**Disentangled Representation Learning for Multi-View Clustering via von Mises-Fisher Hyperspherical Embedding**"

# Requirements

* python 3.8
* cuda
* pytorch
* scipy
* scikit-learn

# How to run
* Specify your arguments in `main.py` : you can change dataset and other arguments there
* run `python train.py`
